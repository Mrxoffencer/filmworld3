Website Bio

This website basically a clone of IMDB Website. It is Made with Open Source API server of IMDB which is a OMDB API in Django, a Python base Web FrameWork to develop website. In this wbsite you can search any movie available on IMDB server, when you search any movie or webseries on this website first it will check its local sever for related querry and retrive resuts to client side, but if any data related to querry does not available in local server then it will check the data related to that querry on Imdb server any retrive data to the client side and at the same time it also save that data in local database, so the next time if anybody try to search same data then the web application retrive that data from the local server rather then check again at OMDB server. Bsically every time when some one try to search new data that is not present in local database then it will update its database with new data. In this website you can also rate any movie and add in your watch list and comment on someone review and many more.
